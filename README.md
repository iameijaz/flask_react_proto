# A Prototype App to use Flask Server and React Frontend App


## Available Scripts

Run the server by going to flask_server folder, and then starting the `server.py` script.

In the project(frontend Folder) directory, you can run:

### `npm run build`
### `npm install -g serve`
### `npm run start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
